# Glow Clinic — AR Business Card

Immersive 3D AR experience for a beauty & skincare clinic.

**Live:** [https://sassongal.github.io/glow-clinic-ar/](https://sassongal.github.io/glow-clinic-ar/)

## Features
- Three.js 3D world with rose gold + white theme
- Floating torus rings, beauty bubbles, skincare bottles, mirror, orbiting crystals
- 400 sparkle particles + falling flower petals
- Interactive parallax camera (touch/mouse)
- Social links: Instagram, WhatsApp, Phone, Website
- vCard contact download
- WhatsApp "Book Now" with pre-filled message
- Print-ready business card design (front + back)

## Tech Stack
- Three.js r128
- Pure HTML/CSS/JS — zero build step
- GitHub Pages deployment
